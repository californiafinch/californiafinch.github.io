# 跨国公司

1. 概念：跨国公司是指由分设在两个或两个以上国家的实体组成的企业，而不论这些实体的法律形式和活动范围如何。
2. 特征：跨国性、战略的全球性、管理的集中性、内部实体的相互联系性。
3. 法律地位：不是国际公法的主体，作为内国法人是国际经济法的主体
4. 实质：多个法律实体，一个经济实体。

# 国际货币基金组织

# 简介

IMF（International Monetary Fund），根据 1944 年 7 月在美国布雷顿森林会议签订的《国际货币基金协定》，于 1945 年 12 月 27 日成立。

# 宗旨

1. 促进国际货币合作
2. 促进国际贸易的扩大与平衡发展
3. 促进汇价稳定
4. 协助建立成员国间经常性交易的多边支付制度
5. 通过贷款调整成员国国际收支的暂时失衡

# 职能

1. 制定规章制度的职能
2. 金融职能

# 世界银行集团

2. 国际金融公司（IFC）
3. 国际开发协会（IDA）
4. 多边投资担保机构（MIGA）
5. 国际投资争端解决中心（ICSID）

# 世界贸易组织

# 简介

WTO（World Trade Organization），根据 1994 年 4 月 15 日在摩洛哥马拉喀什签订的《世界贸易组织协定》于 1995 年 1 月 1 日成立。

# 职能

1. 为《世界贸易组织协定》的实施、管理、运用提供便利，并促进其目标的实现
2. 为成员方就多边贸易关系进行的谈判提供场所
3. 对争端解决谅解规则程序进行管理

# 主要机构

主要机构有部长会议、总理事会、秘书处等。
1. 部长会议是世贸组织的最高权力机构，有权对各多边贸易协定所涉一切问题按法定程序做出决定并在其职能范围内采取行动。
2. 总理事会由所有成员派常驻代表组成，是部长会议休会期间代行其职权的执行机关，执行《世界贸易组织协定》所赋予的各项职能。

# 国际法方面的渊源

# 国际条约

国际条约是国家、国际组织间所缔结的确定其相互关系中的权利和义务的国际书面协议。

# 习惯国际法与国际商事惯例

1. 概念：习惯国际法与国际商事惯例是在国际交往中逐渐形成的不成文的原则和规则。
2. 构成因素：一般认为，构成习惯国际法与国际商事惯例，需要具备两个因素，一个是物质因素，即有重复的类似行为，一个是心理因素，即人们认为有法律拘束力
3. 国际商事惯例的被肯定和被承认：《联合国国际货物销售合同公约》（CISG）第九条规定：“（1）双方当事人业已同意的任何惯例和他们之间确立的任何习惯做法，对双方当事人均有拘束力；（2）除非另有协议，双方当事人应视为已默示地同意他们的合同或合同的订立适用双方当事人已知道或理应知道的惯例，而这种惯例，在国际经贸上，已为有关特定贸易所涉同类合同的当事人所广泛知道并为他们所经常遵守。”
4. 特点：自发形成，由国际民间组织编撰成文；只有经当事人选择，才对特定的当事人具有法律约束力；当事人选择适用时可以任意增、改、删。

# 一般法律原则

一般法律原则通常是指各国法律体系共有的原则。

# 国际组织的决议

1. 国际组织决议的效力，应根据该组织的章程确定。根据《联合国宪章》的规定，安理会对威胁国际和平与安全所作出的决议是具有拘束力的。
2. 对联合国大会决议的效力，根据《联合国宪章》的规定，联大的职权是讨论和建议，因此联大决议一般属于建议性质，不具有法律拘束力。

# 国内法方面的渊源

各国调整涉外经济关系的国内立法形式主要有两种：统一制和分流制。
1. 统一制：制定的国内经济立法均统一适用于涉内与涉外各种经济关系。
2. 分流制：分别制定不同的法律来调整涉内和涉外的经济关系，国内经济法与涉外经济法二者并行。
