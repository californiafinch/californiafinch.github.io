# 金融调控的特征

1. 金融调控主体是中央银行
2. 金融调控以货币政策和宏观审慎政策为双支柱
3. 金融调控必须依法进行
4. 金融调控的目的是稳定币值以促进经济的发展

# 金融调控法的目标

1. 防范和化解系统性金融风险，维护金融稳定
2. 保持社会总需求与总供给的平衡

1. 促进金融服务实体经济原则
2. 间接调控原则
3. 计划指导原则
4. 相互协调原则
