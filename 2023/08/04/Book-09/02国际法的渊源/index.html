# 第一节 国际法渊源

1. 本质是国际法的具体表现形式
2. 国际法渊源是认识和使用国际法的基础

# 第二节 国际法渊源的表现形式

主要包括国际条约、国际习惯和其他辅助性渊源（一般法律原则、司法判例、权威公法学界学说等）

# 国际条约

1. 国际条约是具有缔约能力的国际法主体之间确定的，意思一致的书面协议。
2. 协议以国际法为标准。
3. 国际条约是 20 世纪后最主要的渊源。

# 国际习惯

# 国际习惯的概念

国际习惯是指在国际法律关系中具有法律拘束力的一致性一般国家惯例或通例。

# 国际习惯的地位

国际习惯曾经是国际法最重要的渊源。在现代国际法上，国际习惯仍然是一项主要渊源。

# 国际习惯的构成要素

两个要素说：一是存在一般国家惯例，即国家实践；二是这种惯例被接受为法律，即法律确信或法律必要确信。这两个要素必须同时存在，不可或缺。

# 一般实践

1. 谁的实践：利益方，不要求所有的国家；
2. 持续时间：没有具体的规定，一段时间，足够长；
3. 形态：反复行为；
4. 如何证明：国家语言和行为，法院判决、元首的外交发言等。

# 法律确信

1. 主观要素：难于证明
2. 持续反对者原则：合理性与争议性
3. 速成国际法：合理性与争议性

# 国际习惯的功效

1. 对条约的补充
2. 对法律的补充

# 国际条约与国际习惯之间的关系

1. 静态：与条约并列为国际法渊源的一种；
2. 动态：习惯被编纂为条约；条约可以作为习惯的一部分。

# 一般法律原则

理论上主要有两种解释：一是将它理解为国际法律体系内形成的一般原则；二是将它视为各国法律体系所共有的原则。

这两种类别的一般法律原则的存在得到国家实践、国际条约以及国际性法院或法庭裁判的印证。

适用性与辅助性。

# 司法判例和权威最高之公法学家学说

司法判例和权威最高之公法学家学说不是国际法的渊源，而只是确定法律原则的辅助资料或附属渊源。

普遍性国际法律机构，包括国际法院、常设仲裁法院、国际海洋法法庭和世界贸易组织争端解决机构，不是一个立法机关。

两者的意义：能够澄清、完善和解释国际法规则和促进国际法的发展

# 国际组织的决议

1. 具有一定的特殊性。
2. 本身具有立法权的国际组织，比如联合国安理会；欧盟理事会和委员会。其具有立法权的部门作出的决议。
3. 国际组织自身发展中作出的决议。

# 国家的单方面行为

1. 国家在国际上经常采取意图产生法律效力的单方面行为。
2. 明示和公开方式自发作出的单方面声明。
3. 表现形式可以分为两类：一类是国家借以承担义务的行为，比如承诺、承认或弃权；另一类是国家借以重申权利的行为，比如抗议。

# 第三节 国际法的编纂

# 编纂的概念与发展

1. 国际法的编纂又称国际法的法典化。
2. 官方的编纂是指由政府间国际外交会议或政府间国际组织进行的编纂，是国际法发展的最主要方式。
3. 由政府间国际组织进行编纂是从国际联盟开始。

# 联合国与国际法的编纂

1. 联合国成立后，国际法的编纂进入一个新的发展时期。
2.1947 年联合国大会成立负责编纂工作的主要机构 —— 国际法委员会，并通过《联合国国际法委员会章程》
