# 第一节 国家法基本原则的概念

国际法基本原则是指国际社会公认、具有普遍意义、适用于国际法各个领域并构成国际法基础的法律原则。

# 第二节 国际法基本原则特点及地位

# 国际法基本原则的特点

1. 国际社会公认
2. 具有普遍意义
3. 适用于国际法各个领域
4. 构成国际法整个体系的法律基础
5. 具有强行法性质

# 与强行法的关系

国际法基本原则是国际法规则化产物，是国际社会的宪法性原则。体现了国际法律秩序的基本价值，是国际法体现的法律基础。

# 国家主权原则

# 国家主权的概念

# 国家主权的性质

国家主权是国家最本质的属性，是国家固有的权力。

# 国家主权内容

1. 国家主权内容不断丰富：传统是领土主权，之后发展到自然资源的永久主权。
2. 所有国家一律享有主权平等。
3. 各国均有责任充分并一秉诚意履行其国际义务，并与其他国家和平相处。

# 禁止使用武力或武力威胁原则

# 禁止使用武力或武力威胁原则的确立

禁止使用武力或武力威胁原则的确立是限制、废弃传统战争权的结果。

# 武力的含义

1. 狭义：仅指武装的或军事的力量
2. 广义：它还包括施加政治和经济压力

根据宪章实践，禁止使用武力或武力威胁中的武力是指武装的或军事的力量。
武装力量的构成由该国法律规定。
最严重、最危险的非法使用武力形式也构成侵略。

# 武力与武力威胁的关系

武力与武力威胁为并列关系，禁止使用武力的规则同样适用于武力威胁。

# 禁止使用武力或武力威胁原则的性质

禁止使用武力或武力威胁原则是一项强行法规则，具有习惯国际法地位。国际法院在许多判例中确认禁止使用武力或武力威胁原则。

# 禁止使用武力或武力威胁原则的例外

1. 国家行使自卫权
2. 安全理事会决定或授权使用武力
3. 对第二次世界大战的 “敌国” 使用武力
4. 受殖民地统治的人民行使自决权而使用武力。

# 不干涉内政原则的确立

# 不干涉内政原则的性质

不干涉内政原则是一项习惯国际法和强行法。

# 善意履行国际义务原则的地位

善意履行国际义务原则是一项国际强行法和国际习惯法。

# 善意履行国际义务原则的意义

国际法的有效性和国际秩序的稳定性都依托于各国忠诚遵守国际法的规范并善意履行其义务。

# 第四节 和平共处五项原则

# 和平共处五项原则的内容

1. 互相尊重主权和领土完整
2. 互不侵犯
3. 互不干涉内政
4. 平等互利
5. 和平共处

# 和平共处五项原则的意义

体现了主权、正义、民主、法治、和平共赢等核心价值观，并被不断赋予新的内涵（从 “和平共处” 到 “和平发展” 到 “和谐社会”）。

对中国而言：
1. 是中国外交政策的基石，被纳入中国与一百六十多个国家的建交公报与双边条约中
2. 是中国对国际秩序构建提议的重要方针
3. 载入《中华人民共和国宪法》

对国际社会而言：
1. 在国际社会中广泛适用
2. 载入 1955 年《亚非会议最后公报》
3. 在《国际法原则宣言》《各国经济权利义务宪章》等重要国际文件中体现
