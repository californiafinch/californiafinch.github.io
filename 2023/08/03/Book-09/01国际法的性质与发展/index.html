# 第一节 国际法的重要性

# 国际法的缺陷

1. 法律不能解决所有国际问题
2. 国际法的实效性弱（国际法权威来源于国家的自觉遵守）
3. 实证国际法具有时代局限性（成文法的通病）

# 国际法在国际层面的作用

1. 维系国际秩序
2. 明确国际层面的权利义务

# 国际法在国家层面的作用

1. 规制国家的行为
2. 和国内法形成积极互动

# 国际法对个人的作用

1. 拓宽国际视野
2. 学习国际法有利于通过考试、顺利出国、找到工作

# 学习国际法应当具备的能力

1. 良好的英语阅读能力
2. 批判性思维
3. 理性的大局观

# 国际法拓展阅读参考书目

4.《国际法》沃尔夫刚・格拉夫・魏智通，法律出版社，2002 年
5.《国际法大纲》周鲠生，中国方正出版，2004 年

7.《中国国际法实践与案例》段洁龙，法律出版社，2011 年

# 第二节 国际法的性质与发展

# 国际法的定义

国际法是调整国家之间和国家与非国家行为者之间关系的、具有法律约束力的各种原则、规则和制度的总称。（国家利益的制度化表达）

# 国际法的三个要素

1. 国家利益：所有关于国家的利益
2. 规则手段：和平为主
3. 国际关系：与国家公共利益直接相关的跨国关系

# 国际强行法的概念

国际强行法是指国家之间国际社会全体接受并公认为不允许损抑且仅有以后具有同等性质之一般国际法规律始得更改之规律。

# 国际强行法得特征

1. 一般性与广泛性
2. 不可克减性
3. 优先性

# 国际法的性质

1. 法律性
2. 国际性
3. 普遍性

# 国际法与国内法相比的特征

1. 初级法
2. 协定法
3. 平位法
4. 弱法

# 第三节 国际法的发展及主流学说

# 国际法发展的重要历程

# 古代国际法时期

1. 思想受到神权、宗教、道德思想观念影响
2. 战争为主
3. 无延续性

# 17 世纪：近代国际法

1. 西方形成相对成熟的国际法
2. 中国此时仍是中央帝国，朝贡体系

# 18-19 世纪

1. 条约增多
2. 国际组织壮大

# 20 世纪：现代国际法

1. 倡导以法律规则处理争端
2. 建立国际联盟
3.1919-1939：20 年国际关系紧张

# 二战后

1. 建立国际组织、强化国际规则
2. 联合国建立：人权、平等、和平等原则纳入其中

# 21 世纪：当代国际法

# 国际法的主要流派

# 自然法学派

先验的规则（格劳秀斯为代表学者、二战后二次发展）

# 实证法学派

依据国际社会形成的约定解决问题

# 社会法学派

国际法与一国的外交利益、立场、文化紧密相关

# 国际法的主要学说

# 现实主义理论

国际关系以国家权力为中心

# 自由主义理论

国际关系的本质是国家利益

# 建构理论

国家间的行为与其观念直接相关
