# 股份

# 股份的概念

股份是股份有限公司资本构成的最小单位。

# 股份的特点

1. 不可分性，股份是资本构成的最小单位
2. 金额的均等性，股份是对资本的等额划分
3. 权利上的平等性，股份是股权的基础
4. 自由转让性，股份表现为有价证券

# 股权的概念

狭义的股权是指股东因向公司出资而享有的权利。

# 按股权内容不同，可以分为自益权和共益权

# 股权的行使

# 股权的代理行使

股东既可以亲自行使股权，也可以委托代理人代为行使股权。

# 股东知情权的行使

股东知情权，是指法律赋予股东的通过查阅、复制公司章程、股东会与董事会决议、公司会计报告、公司会计账簿等文件材料，了解公司运营状况和公司高级管理人员业务活动的权利。

# 股东利润分配权的行使

利润分配权，是指股东按照出资或股份比例请求公司分配利润的权利。

# 股权的确认

股权的确认实质上就是对股东资格的确认。主要涉及三种情形：不出资、瑕疵出资和隐名出资。

# 不出资的股权确认

# 瑕疵出资的股权确认

# 出资不足

1. 出资不足，是指虽有出资行为，但未依约足额缴纳出资。
2. 出资不足，依法承担违约责任。

# 出资不实

1. 出资不实，是指虽有出资行为，且形式上已足额缴纳，但用以出资的非货币财产的实际价值显著低于公司章程所定价额。
2. 出资不实，依法承担补足差额责任。

# 抽逃出资

1. 抽逃出资，是指虽有出资行为，且已足额缴纳，但在公司成立后以各种理由非法撤资，从而导致公司注册资本的减少。
2. 抽逃出资，依法承担改正、罚款责任。
